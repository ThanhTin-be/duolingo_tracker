Hướng dẫn nhanh

1. Đăng nhập Duolingo Web trên trình duyệt.

2. Nhấn F12 → Tab Application → Cookies → Sao chép giá trị của jwt_token.

3. Click Thêm Tài Khoản trên Dashboard này, đặt tên gợi nhớ và dán token vào.

4. Data được lưu trữ an toàn trong LocalStorage máy bạn.
